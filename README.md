# Plantcare-AI
PlantCare AI is an intelligent plant monitoring and recommendation system that helps users identify plant health issues and receive smart care suggestions using AI-based analysis. 

# Project Overview
PlantCare AI is an AI-powered system that detects plant diseases using transfer learning techniques and provides smart recommendations for plant care.

# Tech Stack

- Python
- TensorFlow & Keras
- MobileNetV2 (Transfer Learning)
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

# Dataset
PlantVillage Dataset

# Approach
The project uses transfer learning with MobileNetV2 pretrained on ImageNet. 
The final classification layer was replaced and fine-tuned on the PlantVillage dataset 
to detect multiple plant diseases.
